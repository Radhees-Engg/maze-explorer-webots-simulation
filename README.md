# Maze_Explorer_Webots
An Autonomous Maze Explorer simulated in Webots that maps the Unknown grid using DFS(Depth First Search) algorithm. It features PID-controlled navigation for precise movement and return to start functionality.

## Features
* **Autonomous Exploration:** Utilizes DFS algorithm and Sesnors to Autonomously explore the maze and maps the Unknown gird.
* **Precise Movement:** Implements PID control for precise Motor control to execute sharp 90 degree turn.
* **Return to Start:** Once it explores the grid, it's programmed to return to the starting position.
* **Webots Simulation:** Integrated Program and real world physics entirely in Webots environment.

## Language
Language Used: C++. 

## Getting started
* Download and Install Webots from *cyberbotics.com*.
* Ensure you have a C++ compiler.
* Download the MazeExplorer file.
* Open the world and start to create.
